Images and Multimedia

Images and multimedia are something that is a must for a modern website, and to be fair it is relatively easy to be implemented in this tutorial, we will look at the basics for implementing them.

Images

To implement and image we use the <img> tag. This tag is empty which means it does not need a closing tag but it needs an attribute in order to work. The most common attribute is src where we put a copy of the image location it can be a path to a file or an URL.

Example:

                <!DOCTYPE html>
                <html>
                <head>
                <title>Title</title>
                </head>
                <body>
                <img src="images/img_tut1.png">
                </body>
                </html>

Result:


We can also set dimensions of the image, this is possible with the help of height and width attribute, another useful atribute is alt. Alt is used to set a text if the image cannot be loaded.

Example code:

        <!DOCTYPE html>
        <html>
        <head>
        <title>Title</title>
        </head>
        <body>
        <img src="images/img_tut1.png" height="150" 
        width="200" alt="img_tut1">
        </body>
        </html>

Multimedia

You can add audio and video with the help of HTML, it is relatively easy to do that with HTML 5, but unfortunately we are not covering HTML 5 in this tutorial, so we will do it the old-fasioned way. Actually there are several ways, anyway you can use applet tag it allows you to add Java applet to a page. However, the most used ones used are embed and object tags. You can use another parameters when embedding a video or audio, but we are not covering that in depth now.

Example(requires internet connection):

Example:

        <!DOCTYPE html>
        <html>
        <head>
        <title>Title</title>
        </head>
        <body>
        <object data="http://www.youtube.com/embed/Vnt6ktq5UU8" width="550" height="425">
        </object>
        </body>
        </html>
