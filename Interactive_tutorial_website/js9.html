Comments

We know fairly much about JavaScript know, the previous tutorials were kinda brain challange so now let's move to something more easier but extremely important - Commnets
Comments are very important part of the codeing process because you need to know for yourself what each thing does in case you forget and furthermore other people will need guidance when they look at your code, because you may write it but someone else to maintain it, commentig your code well is always a good practice.

There are two types of comments in JavaScript:

Single Line comments

They start with // and can be on the same line as the code, everything after // to the end of the line will be ignored by JavaScript.

Single line comments are used to explain the code mainly. Although they can be used to prevent some lines of executing, this is done when testing usually.

Example

// Change heading:
document.getElementById("soSad").innerHTML = "My First Comment.";
// Change paragraph:
document.getElementById("exOne").innerHTML = "This is fun !";

Here the comments are placed above the code.

Example

var x = 321;      // Declare x, give it the value of 321
var y = x +999;  // Declare y, give it the value of x + 999

This example uses a single line comments after the end of the line, used to describe code.

Example

// document.getElementById("soSad").innerHTML = "My First Comment.";
document.getElementById("exOne").innerHTML = "This is fun !";

On this example we use single line comment to prevent code from execution

Multi Line comments

Multi-line comments start with /* and end with */. Any text/code between /* and */ will be ignored by JS.

Multi line codes are used to describe code but in more detail or when we need to write some info about the script etc. They can also be used to prevent larger amounts of code from executing when testing.

Example

/*
The code below will change
the elements with id = "soSad"
and id="exOne"
in my web page:
*/
document.getElementById("soSad").innerHTML = "My First Comment.";
document.getElementById("exOne").innerHTML = "This is fun !";

This example uses a comment block to prevent execution of multiple lines:

Example

/*
document.getElementById("soSad").innerHTML = "My First Comment.";
document.getElementById("exOne").innerHTML = "This is fun !";
*/

Make sure to comment you code well, this will save you time and effort !
