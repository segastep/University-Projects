CSS Selectors

Selectors are used to select and maniplate HTML elements, based on their id, class, type, attribute, and more. With the help of selectors you can manipulate single or large amount of elements, you can override properties etc. Selectors are one of the most important things when you write CSS style sheet.

The element Selector

The element selector selects elements based on the element name.
For example you can select all <p> elements on this page using this this approach.

Example Code:

		p {
			text-align: center;
			color: red;
		}

This example will align all <p> elements to the center and will make them red.

The id Selector

The id selector uses and id attribute of HTML and it's used to modify a specific element from the HTML.
To select an element with a specific id, write a hash character, followed by the id of the element.

Example Code:

		#heading1 {
			text-align: left;
			color: green;
		}
The class Selector

The class selector is used to select a elements with a specific class attribute. To make a selection of specific class attribute write a period character followed by the name of the class.

Example Code:

		.right {
			text-align: right;
			color:blue;
			margin: 5px
		}
Grouping Selector

If you have elements with the same style definitions you can group them all insead of writing styles for each one separately.

Assuming we have elements with the same style definitions :

		.h1 {
			text-align: center;
			color:blue;
		}
