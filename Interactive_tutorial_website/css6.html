Colors

Backgrounds have a lot of implementations in the building of a website you can use the to give color to your page, <div> tag etc. They are widely used, you can even use images as background of a certain element of the whole page, you can repeat one background multiple times etc.

In this topic we will cover the five most important properties used with backgrounds.

Background Color

Back ground color will set the background of an element in a color you like.

Example:

div {
	background-color: #b0c4de;
}

Result:

Background colors can be specified by using one of the following values:

HEX - example "#ff3286"
RGB - example "rgb(220,100,10,)"
Color name - example "blue"
Background Image

As the name hints we will use an image for a background of our element which in our case will be div. We can repat the image horizontally or vertically depending on our needs or we can not reapeat it at all.

Example:

div {
    background-image: url("img.png");
    background-repeat: repeat-x;
}

Result:

This will repeat the image horizontally, to repat it verically use repeat-y, or not to repeat it at all use no-repeat

Using background properties you can tell your backgrounds where exactly you want it.

Example Code:

{
    background-image: url("img.png");
    background-repeat: no-repeat;
    background-position: right left;
}
