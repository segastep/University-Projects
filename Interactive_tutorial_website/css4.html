Positioning

CSS positioning is very important for your website, it is used to set the positioning rules for the elements in your HTML document. With the help of it you are allowed to position an element in the way you like, you can also put elements behind or infront one another or specify what to happen when the content is too big.

Elements can be positioned using the top, bottom, left, and right properties. However, these properties will not work unless the position property is set first. They also work differently depending on the positioning method.

There are four types of positioning static, relative, fixed and absolute.

Static positioning

All the elements are static positioned by default in HTML e.g they follow the flow of the page and they are not affected by the top, bottom, left, and right properties.

Relative positioning

Relative positioning allows you to specify a specific offset (left, top etc) which is relative to the element's normal position in HTML flow.

Example Code:

div.left {
	position: relative;
	left: -20px;
	background:red;
}

div.right {
    position: relative;
    left: 20px;
}

Example Result:

Relatively positioned elements can be moved and overlap other elements in the page, but the space reserved for the element is still preserved in the norlam flow of the page.


Absolute positioning

When using absolute positioning you have to be extremely careful because your absolute position element is positioned relative to the first parent element that has position other than static, if no such element is found, will be used as a containing block. Absolute positioned elements don't appear in the normal page flow and the document and other elements behave like the absolute position element is not existing. Furhtermore, absolute positioned elements always appear on top of other elements if z-index is not specified. You will see what this means in the next examples.

Absolute positioning example :

div.top{
    position: absolute;
    width: 100px;
    height: 150px;
    color:red;
}

Z-index example:

img {
    position: absolute;
    left: 0px;
    top: 0px;
    z-index: -1;
}

When elements are positioned outside the normal flow, they can overlap other elements. The z-index property specifies the stack order of an element (which element should be placed in front of, or behind, the others).

An element can have a positive or negative stack order.An element with greater stack order is always in front of an element with a lower stack order.
