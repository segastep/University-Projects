Introduction to Loops

Imagine you wanted a script to perform the same routine over and over again for 100 times, would you like to write line of code for each step ? I guess no…
Instead adding 100 lines of mostly equal code, loops can do that for you.

JavaScript has two different kinds of Loops: for and while

For Loop

The form loop is used when we know how many times the script should be perfomed.

Imagine we have an array with names 

var namesArray = ["John", "Rich", “Sue”, “Simon”, “Ben”];

Now we want this names to appear on our web page, you can do it the easy way using loops or the hard way typeing everything by hand.

Bad Example :

name+= namesArray[0] + "<br>"; 
name+= namesArray[1] + "<br>"; 
name+= namesArray[2] + "<br>"; 
name+= namesArray[3] + "<br>"; 
name+= namesArray[4] + "<br>";

Well it will work but do we really need to do it this way ?

Example using loop:

for (i = 0; i < namesArray.length; i++) {
    name += namesArray[i] +     "<br>";
}

The for/in loop

The statement of this loop goes through the properties of an object, remeber that arrays in JS are objects right?

Example

var cars = {carName:"Ford", carModel:"Mustang", productionDate:2010};
var person = {5};
var carsPrint = "";
var x;
for (x in cars) {

While Loops

While loops are used when we don't know how many times we need to execute the code but we know when to stop executeing it (usually we have a condition which have to become false before the loop ends)

Syntax

while (condition) {
	code block to be executed
}

Example

In this example, the loop will be executing over and over again until the variable (i) is less or equal to 5:

while (i =< 5) {
    text += "The number is " + i;
    i++;
}
Do/While loop

This loop will execute the code once after that will check if the condition is met and it will execute the code over and over again as long as the condition is true

Syntax

do {
    code block to be executed
}
while (condition);

Example

In this example, the loop will be executing over and over again until the variable (i) is less or equal to 5:

do {
    text += "The number is " + i;
    i++;
} while (i < 10);

Don't forget to increase the variable otherwise the loop will continue forever and eventually crash your computer, you can use and booleans as condition.
