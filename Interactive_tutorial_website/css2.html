Units

CSS has several different units for expressing a length. Many CSS values take length properties. For some CSS properties we can have a negative values for length.

There are two types of units - absolute and relative.

Absolute units

Absolute units speify a fixed lenght and they are not the best choose when you create a site wich will be used on different screen sizes because they do not rescale for the specific screen size.

Unit Description
cm centimeters
mm millimeters
in inches
px pixels
pt points(1pt = 1/72 of 1in)
Relative Units

Absolute units specify a fixed length. Their use is not recommended if the website is going to be seen from different screen sizes, as it does not rescale for the specific screen.

Unit Description
em Relative to the font-size of the element
ex Relative to the x-height of the current font
ch Relative to width of the "0"
rem Relative to font-size of the root element
vw Relative to 1% of the width of the viewport
vh Relative to 1% of the height of the viewport
vmin Relative to 1% of viewport's smaller dimension
vmax Relative to 1% of viewport's larger dimension
%

A well designed page uses both types.
