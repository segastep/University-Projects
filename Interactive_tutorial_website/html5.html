Inline Tags

Inline tags occupy only the space bounded by the tags that define the inline element, and they must be put inside a block-level element. A typical example of inline tags is the <strong> tag, which gives strong emphasis like this.

Here are some of the most lidely used Inline tags:

Tag Visual appearance
<em> Text in em
<strong> Text in strong
<code> Text in code
<abbr> and <acronym> HTML
<b> Text in bold
<i> Text in italics
<big> Text in big
<small> Text in small
<sub> Text in sub
<sup> Text in sup
<u> Text in u
<s> Text in s
