Paragraph Break and Rules

Text in HTML is almost always displayed with the paragraph tag, like this text you are reading right now. It's a block-level element so it begins on a new line every time. To use it you just need an opening <p> tag, some text, and a closing </p> tag.

But sometimes you just want a new line without creating a new paragraph, and the <br /> is just for that. It does not need a closing tag, so that's cool. But it is not recommended the use of it very much, only if you really need it.

And then there are horizontal rules to seperate the text, like i just did now. To use them just use the <hr /> tag, and the best thing is that it does not need a closing tag too!

In the example below you can see all of the tags from this tutorial.

Example code:

                  <!DOCTYPE html>
                    <html>
                    <head>
                    <title>Title</title>
                    </head>
                    <body>
                    <p>This is a normal paragraph</p>
                    <p>This is a <br /> paragraph with <br /> line breaks</p>
                    <hr />
                    <p>This paragraph is under a horizontal rule</p>
                    </body>
                    </html>
