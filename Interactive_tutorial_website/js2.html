JavaScript Where to?

JavaScript can be place in the <head> and in the <body> part of a page. But the content have to be surrounded by the <script></script> tag. In some older examples <script type="text/javascript"> attribute may be used, but now the type attribute is not required because JS is the default scripting language for HTML.

The <script> Tag

Example

<script>
document.write("<h1>Hello World</h1>");
</script>

Result

Hello World
JavaScript: Where on the page?

It is always a good idea to put as much of your JS in the <head> of your HTML document(function definitions etc)

Example

<html>
<head>
<title>Example</title>
<script type="text/javascript">
function hello(){
alert("Hello World");}
</script>
</head>
<body>
<script type="text/javascript">
document.write("<h1>Some Scripting</h1>");
</script>
<div> <input type="button" value="Click here" onclick="hello();"/> </div>
</body>
</html>

Result

External JS

Scripts can also be placed in external files. External scripts are practical when the same code is used in many different web pages. JavaScript files have the file extension .js.
To use an external script, put the name of the script file in the src (source) attribute of the <script> tag:

Example:

<!DOCTYPE html>
<html>
<body>
<script src="firstScript.js"></script>
</body>
</html>
