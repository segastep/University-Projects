What is CSS?

CSS is an abbreviation standing for Cascading Style Sheets. This is the magic that you add to HTML to make it look good and pretty. CSS is list of rules for each component of you HTML file, in fact CSS can be used with any mark up language.

There are several versions of CSS, the newest one CSS 3 which is so powerful that you can do almost anything with it even animations.

Here you can see some more information about the CSS versions:

CSS Version Year
CSS 1 December 17, 1996
CSS 2 May 1998
CSS 2.1 7 June 2011
CSS 3 June 2012
