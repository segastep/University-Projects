What is JavaScript?

Introduction

JavaScript is one of the most popular programming languages. JavaScript was created back in the 1995. It is most commonly used as part of web browsers, whose implementations allow client-side scripts to interact with the user, control the browser, communicate synchronously, and alter the document content that is displayed.

JavaScript is also used in environments that aren't web-based, such as PDF documents, site-specific browsers, and desktop widgets.
