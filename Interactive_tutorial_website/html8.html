Tables

HTML tables are used, when you want ot arrange content(text, images, links etc.) into rows and columns. To create a table in HTML we use the <table> tag, in which we use the <tr> tag to create rows, and the <td> tag to create data cells.

Here is an example of a simple HTML table:

Example code:

                    <!DOCTYPE html>
                    <html>
                    <head>
                    <title>Title</title>
                    </head>
                    <body>
                    <table border="1">
                    <tr>
                    <td>Cell 1 Row 1</td>
                    <td>Cell 2 Row 1</td>
                    </tr>
                    <tr>
                    <td>Cell 1 Row 2</td>
                    <td>Cell 2 Row 2</td>
                    </tr>
                    </table>
                    </body>
                    </html>

Example Result:

Cell 1 Row 1 Cell 2 Row 1
Cell 1 Row 2 Cell 2 Row 2

To add a heading to the columns, we use the <th> tag in the top row of our table.

Here is a simple example:

Example code:

                    <!DOCTYPE html>
                    <html>
                    <head>
                    <title>Title</title>
                    </head>
                    <body>
                    <table border="1">
                    <tr>
                    <th>Column 1</th>
                    <th>Column 2</th>
                    </tr>
                    <tr>
                    <td>Cell 1 Row 1</td>
                    <td>Cell 2 Row 1</td>
                    </tr>
                    <tr>
                    <td>Cell 1 Row 2</td>
                    <td>Cell 2 Row 2</td>
                    </tr>
                    </table>
                    </body>
                    </html>

Example Result:

Column 1 Column 2
Cell 1 Row 1 Cell 2 Row 1
Cell 1 Row 2 Cell 2 Row 2

There are some other useful tags and attributes that you can use, which i am going to list them in the table below. But the table below will be styled with CSS, so you can see what can a table look like with some simple CSS.


Tag/Attribute Use
<caption> Adds a caption to the table
border Sets the table border
width Display width of table
cellspacing Sets the distance between cells
cellpadding Sets the padding of the cells
Cell Attributes Use
colspan Number of columns cell should span
rowspan Number of rows cell should span
scope Direction in which a cell provides header information
summary Provides a text summary of table purpose/contents
