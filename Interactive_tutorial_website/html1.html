What is HTML?

HTML or Hyper Text Markup Language is a computer language devised to allow website creation. It's the primary block of the web, without it would not be able to show content on the web. It is relatively easy to learn, with the basics being accessible to most people in one sitting. And it is quite powerful in what it allows you to create.

Since it's creation, HTML has been constantly changing and upgrading. Here is a brief timeline:

HTML Version Year
HTML 2.0 November 24, 1995
HTML 3.2 January 1997
HTML 4.0 December 1997
HTML 4.01 December 1999
HTML5 October 2014

There are some strict versions of HTML called XHTML, which do not let you to make syntax errors. Which was good in the past, because that ensured that your website would be the same on all browser. But it is rarely used these days, as browser are much more powerful and can handle syntax errors better.

Here is a sample HTML code:

                        <!DOCTYPE html> 
                        <html> 
                        <head> 
                        <title>This is a title</title> 
                        </head> 
                        <body> 
                        <p>Hello world!</p> 
                        </body> 
                        </html>
