Datatypes

JavaScript has five main data types and in this tutorial we will see how to use them and when to use them, don't get confused at first by the end of this tutotial it will all make sense to you.

Main JavaScript data types are: String, Number, Boolean, Array, Object

JavaScript variables can hold many data types: numbers, strings, arrays, objects and more:

Example:

var length = 6;                            // Number
var lastName = "Steve";                   // String
var cars = ["Audi", "BMW", "Honda"];        // Array
var x = {firstName:"Georgi", lastName:"Nikolov"}; // Object

In programming, data types is an important concept. It is important to have some knowledge about the type, to be able to operate onvariables.

Withot data type, computer will sometimes get confused and won't be abe to solve some equations for example:

var x = 5 + "BMW"

It does makes sense to you that BMW is a string and 5 is an integer, but since JavaScript is not a person it would treat both of them as strings and will eventually produce the following result:

5BMW

What if we have something like that:

var x = 5+6 +"BMW"

As JS evaluates expressions from left to the right. Different sequences can produce different results, back to the previous example, the result would be:

Result:

11BMW

What if we swap places of the BMW and the number ?

JavaScript:

var x = "BMW" + 5 +6 ;

Result: 

BMW56

In the first example JavaScript trats 5 and 6 as numbers, until it goes to "BMW" which is a string. But in the second example since the first operand is a string, all other operands are treated as strings as well.

Dynamic Types

Dynamic types in JS means that the same variable can be used as different types:

Example

var x;  // X is undefined
var x = 1;  // X is defined as number
var x = "Foo"; //  X is defined as a string
JS String

In JS strings are written in quotes and you can use single or double quotes.

var cityName = 'London' // Using double quotes
var cityName = 'Sofia'     // Using single quotes

You can also use qoutes inside a string, but the must not match the quotes surrounding the string

Example

var response = "Sure we can do it!'";  //Single quote inside a double quotes
var response = " Her name is 'Sarah'";  // Single quotes inside double quotes
var response = ' Her name is "Sarah"';  //Double quotes inside singe quotes
JS Numbers

Unlike other languages JS has only one type of numbers. You decide how to write them

Example

var x1 = 21.00;  // Written with decimals
var x2 = 21;     // Written without decimals

You can use scientific notations to represent too big or too small numbers

Example

var y = 123e5;      // 12300000
var y1 = 123e-5;     // 0.00123
JS Arrays

In JS arrays ar written with square brackets, the array elements are separated by commas.

Example

var beers = ["Budwiser", "Haineken", "Corona"];

Like in other programming languages array start counts from zero so Budwiser will have position 0 and Haineken will have position 1 etc.

JS Objects

Objects in JS are written with curly braces. Object properties are written as name: value pairs, separated by commas.

var person = {firstName:"Neil", lastName:"Speirs", age:30, hairColor:"black"};
The typeof Operator

You can use the JavaScript typeof operator to find the type of a JavaScript variable:

Example

typeof "Rich"                // Returns string
typeof 2.99                  // Returns number
typeof true                // Returns boolean
typeof [1,2,3,4]             // Returns object
typeof {name:'Rich', age:22} // Returns object

Undefined and Empty variables

In JavaScript, a variable without a value, has the value undefined. The typeof is also undefined.

Example

var person;               // Value is undefined, type is undefined

Any variable can be emptied, by setting the value to undefined. The type will also be undefined.

Example

person = undefined;       // Value is undefined, type is undefined

An empty value has nothing to do with undefined.
An empty string variable has both a value and a type.

Example

var car = ""; // The value is "", the typeof is string

Null

In JavaScript null is "nothing". It is supposed to be something that doesn't exist. In JS null is object unlike in other languages. You can empty an object by setting it to null:

Example

var person = null;        // Value is null, but type is still an object

You can also empty an object by setting it to undefined:

Example

var person = undefined;  // Value is undefined, type is undefined

Difference Between Undefined and Null

typeof undefined           // undefined
typeof null                // object
null === undefined         // false
null == undefined          // true
