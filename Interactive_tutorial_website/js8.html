Functions

JavaScript Functions are blocks of reusable code, that you can use anywhere in your programme. Using the functions you can devide a big program to smaller parts. Another plus of the functions is that they help you to write modular code. JavaScript is rahter sophisticated language this means JS has all the features to write modular code using functions.

JavaScript Function Syntax

Before we proceed let's take a closer look at functions syntax, make se you understand this well before you proceed !

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

Function names can contain letters, digits, underscores, and dollar signs (same rules as variables).
The parentheses may include parameter names separated by commas: (parameter1, parameter2, ...)

The code to be executed, by the function, is placed inside curly brackets: {}

function name(parameter1, parameter2, parameter3) {
    code to be executed
}

Function parameters are the names listed in the function definition.
Function arguments are the real values received by the function when it is invoked. Inside the function, the arguments are used as local variables.

Functions are invoked on a click, self invoked or when they are called somewhere.
