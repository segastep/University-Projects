Arrays

In JS like in other programming languages arrays are used to store multiple values in a single variable. Arrays are pretty useful in this tutorial we will try to teach you the basics of using arrays. Arrays save time and effort remember it !

The first thing we will learn is how to display an array, in this tutorial we will dipslay arrays inside a <p> element as this is the most common situation.

Example

				<p id="array"><p>
				<script>
				var names = [ "Rich", "Jamie", "Sue"];
				document.getElementByID("array").innerHTML = names;
				</script>

The first line (in the script) creates an array named names. The second line "finds" the element with id="array", and "displays" the array in the "innerHTML" of it.

What is an Array ?

Array is a special variable, which can hold more than one value at a time, imagine you want to declare 30 names, instead of declareing each name as a variable which is time consumeing you can use an array.

So array is a variable that can consist of number of values with one and a same types of data or different.

You can acess each element in the array by looping around the array as each component in the array has a specific position, but we will talk about that later....

Creating an array:
var array-name = [item1, item2, ...];

This is the easiest way it's called array literal

JS Array can be declared using this approach:
var cars = new Array("Rich", "Sue", "James");

Access the Elements of an Array

You refer to an array element by referring to the index number.
This statement accesses the value of the first element in cars:

var name = name[0];

This statement modifies the first element in cars:

name[0] = "Rich";

Objects in an Array

JS variables can be objects. Arrays are special kind of objects, due to this you can have variables of different types in same Array. In JS array we can have orher arrays, objecs or functions.

				myArray[0] = Date.now;
				myArray[1] = myFunction;
				myArray[2] = myCars;

Arrays are special type of object, if use typeof operator on an Array in JS it will return object for array, but yet they are best described as arrays because they use numbers to acess it's elements.

In this example name[0] returns Rich

var name = [ "Rich", "Sue", "James"]

Objects use names to acess its "members". In this example person.FirstName returns Rich

var person = {firstName:"Rich", lastName:"Jones", age:32};
Arrays properties and methods

The real strength of JavaScript arrays are the built-in array properties and methods:

Examples

var x = cars.length;         // The length property returns the number of elements in cars
var y = cars.sort();         // The sort() method sort cars in alphabetical order

The lenght property

The lenght property returns the number of elements in the array

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.length;                    // the length of fruits is 4
Adding Array Elements

The easiest way to add a new element to an array is to use the length property:

Example

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits[fruits.length] = "Lemon";  // adds a new element (Lemon) to fruits

Adding elements with high indexes can create undefined "holes" in an array:

Example

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits[10] = "Lemon";             // adds a new element (Lemon) to fruits
Looping Array Elements

The best way to loop through an array, is using a "for" loop:

Example

var index;
var fruits = ["Banana", "Orange", "Apple", "Mango"];
for (index = 0; index < fruits.length; index++) {
}
